## Key Learnings
1. **PowerShell stderr Trap:** `cmd /c "npx vitest ..."` in PowerShell gibt Exit Code 1 bei stderr-Ausgabe zurück, obwohl Vitest selbst 0 zurückgibt. Lösung: `2>nul` oder `EXIT_CODE=$LASTEXITCODE` prüfen.
2. **Prisma Mock Typing:** `vi.mock('@prisma/client', () => ({ PrismaClient: function(...) {...} }))` erzeugt in `tsc --noEmit` `Awaitable<Partial<unknown>>` Fehler. Workaround: `// @ts-nocheck` in reinen Test-Dateien mit komplexen Mocks.
3. **Agent Timeout Management:** Agent-3 (turbomed+medistar+xisynet) hat sich in endlosen ReadFile/Shell-Zyklen verfangen (mehrere Fehlversuche, dann success). Timeout von 900s war knapp.
4. **Deploy Preflight Timeout:** `npx prisma migrate status` hängt unbegrenzt, wenn DB nicht erreichbar. Timeout-Parameter zu `execFileSync` hinzugefügt (15s für Prisma-Status).
