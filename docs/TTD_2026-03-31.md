## Required E2E run order once infra is healthy

1. Point `DATABASE_URL` at the Supabase pilot database or a healthy local Postgres.
2. Run:

```bash
npx prisma migrate deploy
npx prisma db seed
```

3. Run smoke E2E:

```bash
npx playwright test e2e/critical-path.spec.ts --project=chromium
```

4. Run dashboard slices:

```bash
npx playwright test e2e/arzt-dashboard.spec.ts --project=chromium
npx playwright test e2e/mfa-dashboard.spec.ts --project=chromium
```

5. Run the full suite:

```bash
npm run test:e2e
```
