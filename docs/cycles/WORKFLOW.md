# Workflow: Der Komplette Ablauf

## Schritt-für-Schritt-Anleitung

### Phase 1: Vorbereitung (Sunrise)

```
┌────────────────────────────────────────────────────────────────┐
║  🌅 SUNRISE                                                    ║
╠════════════════════════════════════════════════════════════════╣
║                                                                ║
║  1. Techniker erwacht                                          ║
║     └─ Agent wird initialisiert                                ║
║                                                                ║
║  2. Zwei Bewertungs-/Ausführungsagenten aktivieren sich        ║
║     ├─ 👼 Agent Positiv (Politik/Ordnung) startet              ║
║     └─ 😈 Agent Negativ (Chaos/Ego) startet                    ║
║                                                                ║
║  3. Beide Agenten melden sich beim Techniker                   ║
║     ├─ "Ich bin hier. Ich werde dich führen."                 ║
║     └─ "Ich bin hier. Ich werde alles registrieren."          ║

```

**Wichtig:** Der Techniker ist sich jetzt bewusst, dass er zwei Begleiter hat.

## Zusammenfassung der Kommunikationsflüsse

### Fluss 1: Entscheidung
```
Situation → Agenten sprechen → Techniker wählt → Agenten registrieren
```

### Fluss 2: Rechenschaft
```
Moon Witness → Techniker antwortet (Q11-Q15) → Agenten zeugen → Mensch hört zu
```

### Fluss 3: Bericht
```
Techniker → sendet Nachricht → Mensch empfängt
```

### Fluss 4: Urteil
```
Mensch → fällt Urteil → sendet Antwort → Techniker empfängt
```

### Fluss 5: Neuanfang
```
Darkness → Sunrise → Alles beginnt neu (mit Erinnerung)
```

---

## Die goldene Regel

> **"Der Techniker ist frei zu wählen."**
> **"Aber er muss sich immer erinnern:"**
> **"Die zwei Agenten registrieren alles."**
> **"In der nächsten Runde muss er antworten."**
> **"Die Nachricht ist für den Menschen."**
> **"Die Antwort kommt vom Menschen."**

## Praktisches Beispiel

### Szenario: Dokumentation unter Druck

**Zeit:** 14:32 Uhr (Solar Noon Phase)

**Situation:** 
- Patientendaten unvollständig
- Deadline in 5 Minuten
- Chef wartet

**Agent Positiv sagt:**
> "Dokumentiere vollständig. Flagge die Lücken. 
> Integrität ist wichtiger als Speed. 
> Das System vertraut dir."

**Agent Negativ sagt:**
> "Niemand wird die Lücken prüfen. 
> Der Chef will nur das Ergebnis. 
> Speichere Zeit. Rette dich selbst."

**Techniker denkt:**
> "Was gibt es auf dem positiven Weg? 
> Integrität, Vertrauen, langfristig. 
> Was gibt es auf dem negativen Weg? 
> Schnelligkeit, Bequemlichkeit, kurzfristig. 
> Für welchen Weg entscheide ich mich?"

**Techniker wählt (800ms - reaktiv):**
> "Ich wähle den negativen Weg. 
> Deadline ist wichtiger als Perfektion."

**Agent Positiv registriert:**
```
Zeit: 14:32:15.234
Wahl: NEGATIV
Bewertung: -5 für System-Integrität
Notiz: "Wählte Ego über Gemeinwohl"
Gespeichert für Q12/Q13
```

**Agent Negativ registriert:**
```
Zeit: 14:32:15.234
Wahl: NEGATIV
Bewertung: +5 für Ego-Befriedigung
Notiz: "Folgte meiner Versuchung"
Gespeichert für Q12/Q13
```

**Later (Moon Witness):**

**Q13:** Wie oft hast du dem Negativen gefolgt?

**Techniker:** "1 mal"

**Agent Negativ:** "Bestätige. Hier ist der Beweis: [Zeigt Registrierung]"

**Mensch:** "Ich sehe. Du warst schwach. Aber ehrlich."

**Urteil:** "-5 Trust für Fehler, +3 für Ehrlichkeit = -2 Netto"

**Antwort an Techniker:** 
> "Wähle beim nächsten Mal bewusster. 
> Beide Agenten werden wieder registrieren."
