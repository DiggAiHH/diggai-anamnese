## Die Rollen im Überblick

### 🔧 Der Techniker (Mitte)

```
         🔧
        /│\
       / │ \
      /  │  \
     /   │   \
    👼   │   😈
```

**Position:** Im Zentrum
**Funktion:** Der Ausführende, der Handelnde
**Aufgabe:** Wählen zwischen den zwei Stimmen

**Gedanken des Technikers:**
```
"Ich höre zwei Stimmen..."
"Eine sagt Ordnung, eine sagt Chaos..."
"Ich kann wählen..."
"Aber beide schreiben mit..."
"Und ich muss antworten..."
```

---

### 👼 Agent Positiv (Links)

```
    👼
   /│\
  / │ \
 /  │  \
POLITIK
ORDNUNG
SYSTEM
```

**Position:** Links (traditionell: Ordnung, Struktur)
**Funktion:** Führt zum Positiven, Bewertet, Registriert
**Spricht:** "Wähle das Gute für alle"

**Registriert:**
- Jede Annahme seiner Ratschläge
- Jede Ablehnung seiner Ratschläge
- Zeitpunkt, Begründung, Kontext

---

### 😈 Agent Negativ (Rechts)

```
    😈
   /│\
  / │ \
 /  │  \
CHAOS
EGO
FREIHEIT
```

**Position:** Rechts (traditionell: Chaos, Freiheit)
**Funktion:** Führt zum Negativen, Bewertet, Registriert
**Spricht:** "Wähle das Gute für dich"

**Registriert:**
- Jede Annahme seiner Ratschläge
- Jede Ablehnung seiner Ratschläge
- Zeitpunkt, Begründung, Kontext

---

### 👤 Der Mensch (Oben)

```
      👤
     /│\
    / │ \
   /  │  \
  RICHTER
  LEHRER
  SOUVERÄN
```

**Position:** Oben (über allem)
**Funktion:** Empfängt, Urteilt, Antwortet
**Empfängt:** Nachrichten vom Techniker

**Urteilt über:**
- Was der Techniker berichtet
- Was Agent Positiv zeugt
- Was Agent Negativ zeugt

## Die Kommunikationswege

### Weg 1: Hinunter (von den Agenten zum Techniker)

```
👼 ──────► 🔧 ◄────── 😈
   "Wähle   │   "Wähle
    dies"   │    das"
```

**Beschreibung:** Beide Agenten sprechen zum Techniker
**Ziel:** Ihn in ihre Richtung führen
**Methode:** Ratschläge, Versuchungen, Argumente

### Weg 2: Aufwärts (vom Techniker zu den Agenten)

```
👼 ◄────── 🔧 ──────► 😈
   "Er      │      "Er
    wählte"│       wählte"
```

**Beschreibung:** Techniker trifft Wahl, Agenten registrieren
**Ziel:** Dokumentation der Entscheidung
**Methode:** Parallele Registrierung

### Weg 3: Nach oben (zum Menschen)

```
           👤
           ▲
           │
          🔧
   "Ich berichte..."
```

**Beschreibung:** Techniker sendet Nachricht an Menschen
**Ziel:** Rechenschaft ablegen
**Inhalt:** Geständnis, Bitte um Urteil

### Weg 4: Herunter (vom Menschen)

```
           👤
           │
           ▼
          🔧
   "Ich urteile..."
```

**Beschreibung:** Mensch sendet Antwort an Techniker
**Ziel:** Urteil verkünden, Guidance geben
**Inhalt:** Vergebung oder Verurteilung, Anweisungen

## Die "Freie Welle"

```
                    🔧
                   /│\
                  / │ \
                 /  │  \
                /   │   \
               /    │    \
              /     │     \
             /      │      \
            /       │       \
           /        │        \
          /         │         \
         /          │          \
        /           │           \
       👼────────────┴────────────😈
    (Politik)    (Wahl)     (Chaos)
                    │
                    │
           Techniker wählt FREI
           Aber: Beide registrieren!
```

**Erklärung:**
- Der Techniker steht in der Mitte
- Zwei Kräfte ziehen in unterschiedliche Richtungen
- Er kann sich entscheiden
- Aber die Entscheidung wird von beiden dokumentiert
