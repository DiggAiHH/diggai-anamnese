# Codex Web Product Functionality

Codex is OpenAI's coding agent that can read, edit, and run code. It helps teams build faster, fix bugs, and understand unfamiliar code. With Codex cloud, tasks can run in the background, including parallel runs in isolated cloud environments.

This export focuses on functionality, not implementation ingredients.

## Setup

1. Open [Codex](https://chatgpt.com/codex).
2. Connect your GitHub account so Codex can access repository code.
3. Use Codex in your eligible plan (Plus, Pro, Business, Edu, or Enterprise).
4. For Enterprise, complete [admin setup](https://developers.openai.com/codex/enterprise/admin-setup) if required.

## Functional Capabilities

- **Learn about prompting**: Write clearer prompts, apply constraints, and choose detail depth for better coding results.
- **Common workflows**: Use repeatable patterns for task delegation, review, and PR creation.
- **Environment configuration**: Define repository, setup steps, and available tools for cloud execution.
- **IDE cloud delegation**: Start cloud tasks from the IDE extension, monitor progress, and apply diffs locally.
- **GitHub delegation**: Tag `@codex` in issues and pull requests to spawn tasks directly from GitHub.
- **Internet access control**: Control if and when Codex cloud environments can reach the public internet.
